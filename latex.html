Cost function
\begin{displaymath} J(\theta) = \frac{1}{2m} \sum_{i = 1}^{m} \left( h_{\theta} ( x^{(i)} ) - y^{(i)} \right)^{2} \end{displaymath}
Gradient Descent
\begin{displaymath} \theta_{j} = \theta_{j} - \alpha \frac{1}{m} \sum_{i=1}^m \left( h_{\theta} (x^{(i)}) - y^{(i)} \right) x_{j}^{(i)} \end{displaymath}
